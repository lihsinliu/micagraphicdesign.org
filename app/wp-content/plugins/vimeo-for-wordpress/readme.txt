== Description ==

Awesome features:

* Simple to set up. Just enter your Vimeo username
* Adds a box to the bottom of the post editing page
* Use the dropdown boxes to add your videos or your friends
* You can also manually enter any Vimeo address
* Edit all the parameters of the video player with ease

= This plugin does not work with the Visual (tinyMCE) editor = 


== Installation ==

1. Unzip into your `/wp-content/plugins/` directory. If you're uploading it make sure to upload
the top-level folder. Don't just upload all the php files and put them in `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Click on the 'Vimeo for Wordpress' link under the settings sidebar, add your Vimeo username. 
4. That's it!



== Frequently Asked Questions ==
 = Q. How do I use this plugin? = 
A. Once it's installed add your Vimeo username, then when your write or edit a post a box will appear under the text input box.
From this box you can access you and your friends videos or manually enter the Vimeo URL of a video you want to use.

== Screenshots ==
1. Add your Vimeo username in the settings panel
2. When you go to write or edit a ppost/page a box will appear under the editor, using the dropdown lists you can select you or your friend's videos. Press add video to add the embed code.
3. Be amazed as it adds the code inline
4. If you want to add a video that doesn't belong to you or your friends, you can always copy the URL of the video
5. Copy and paste the URL into the manual entry box and BAM! The plugin takes the data out and creates the embed tag.
6. Press publish and you have Vimeo videos in your posts, hassle free.
